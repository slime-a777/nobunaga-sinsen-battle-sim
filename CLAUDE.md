## Repository Purpose

モバイルゲーム **信長の野望 真戦** (Nobunaga's Ambition: Shinsen) 向けの**戦闘シミュレーター（`battle_simulator.html`）の開発・改良**を主目的とするプロジェクト。あわせてゲームデータの個人参照メモも管理する。

## メインプロジェクト：battle_simulator.html

`battle_simulator.html` は単一ファイル完結のブラウザ動作シミュレーター。

### 現在の機能
- 味方・敵それぞれ武将3名の編成入力
- 戦法の選択（固有戦法・伝授戦法・追加戦法）
- ターン制の戦闘シミュレーション実行
- HPバー表示・戦闘ログ出力
- ピックアップログ（特定ターンの詳細表示）

### 技術スタック
- 単一HTMLファイル（HTML + CSS + JavaScript）
- 外部依存：Google Fonts（Noto Serif JP、Zen Antique）のみ
- フレームワーク不使用

### 改良方針
- ゲームの実際の仕様（戦法発動確率・効果計算式・兵種補正など）への精度向上
- UI/UX の改善
- 新機能の追加（例：複数回試行の統計、編成保存など）

## Game System Notes

- **部隊編成**：武将3名で1部隊。コスト上限あり（序盤15、進行で増加、最大20）
- **戦法種類**：固有（変更不可）／伝授（戦法化で継承）／事件（特定武将を消費）
- **習熟度**：30%〜100%。数値は「習熟30% → 習熟100%」の順で記載
- **評価基準**：SS > S > A の順
- **戦法発動順序**：同一武将の戦法は受動→兵種→指揮→能動→通常攻撃→突撃の順に発動する。同種の戦法は固有→覚醒前伝授→覚醒後伝授の順に発動する。

## Checklist Format

```markdown
| [x] | 武将名 | コスト | 勢力 | 役割 | 凸数:N | メモ |
| [x] | 戦法名 | 種類 | 発動確率 | 伝授元 | 習熟:N | メモ |
```

凸数は `0`〜`5`、習熟は `未` または `30`〜`100`。
