# 信長の野望 真戦 武将データまとめ

> 最終更新：2026年4月8日　参照元：Game8攻略Wiki（game8.jp/nobunaga-shinsen）
> ※ 伝授戦法は「その武将を戦法化したときに得られる戦法」を記載。
> ※ 特性はLv50時のもの。各武将の特性は「特性名（凸数）: 効果」の形式で記載。
> ※ **Lv50属性**はGame8攻略Wikiの「基本属性値と上昇値から算出した値」を記載。実際の表示と若干ズレる場合があります。
> ※ Lv50属性が未記載の武将（帰蝶・妻木煕子・寿桂尼・浅井長政・酒井忠次・伊達晴宗・甘粕景持・宇佐美定満・板垣信方・甘利虎泰・毛利隆元・仙石権兵衛・安東愛季）は取得中。順次更新予定。
> ※ 特性の凸数は多くの場合「0凸・1凸・3凸・5凸」の4段階。一部武将（山県昌景など）は「0凸・1凸・3凸・4凸」。

## ゲーム基本システム

- **部隊編成**：3人の武将で1部隊を構成
- **コスト上限**：序盤は14。凸や進行に応じて上昇
- **勢力ボーナス**：同一勢力3人で全ステータス約10%上昇
- **兵種適性**：S/A/B/C の4段階。S=120%、C=70%
- **戦法種類**：固有戦法（武将固有・変更不可）／伝授戦法（戦法化で他武将に継承可）／事件戦法（特定武将を消費して入手）
- **特性**：武将固有のパッシブ強化。兵種レベル上昇・ステータス%上昇など

## おすすめ編成例

### スタダ（序盤）定番編成
```
お市（コスト3）＋ 蜂須賀小六（コスト5）＋ 浅井長政（コスト6）　合計コスト：14
```
お市の回復で継戦力を確保しつつ低コストで序盤の土地占領を狙う。お市は最安160円、浅井は最安1,600円で必ず入手可能。

### 武田騎馬隊編成
```
武田信玄（補助・大将）＋ 山県昌景（兵刃）＋ 馬場信春（兵刃）
```
武田勢力ボーナス＋赤備え会心特化。風林火山が攻守すべての局面をカバー。

### 立花道雪麻痺コンボ編成
```
立花道雪（兵刃・大将）＋ 伊達晴宗（兵刃）＋ 補助/回復武将
```
電光雷轟で麻痺付与→霹靂一撃で会心コンボ。長宗我部元親の鬼若子で連撃を付与すると麻痺がさらに安定。

### 深慮遠謀耐久編成
```
北条氏康（補助・大将）＋ 宇佐美定満（計略）＋ 回復武将
```
3T被ダメ軽減＋鉄壁＋回復で非常に粘り強い。

### 豊臣回復コンボ編成
```
豊臣秀吉（補助・大将）＋ ねね（回復）＋ 加藤清正 or 竹中半兵衛
```
