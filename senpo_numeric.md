## 5. 計算式

### 兵刃ダメージ

#### ステップ1: 基礎値 (baseDmg)

baseDmg = max(最小保証, 最小保証 + hpDmg(兵力) + attrD(武勇差))

武勇差 : 攻撃側の武勇 − 防御側の統率
attrD(差) : (0.0005 × 差² + 0.9 × 差 + 4.5) × 1.44  
hpDmg(兵力) : テーブル補間値（10,000兵力 → 387）
最小保証 : max(5, floor(兵力/1000) × 2)（10,000兵力 → 20）

#### ステップ2: 戦法ダメージ

戦法ダメ = baseDmg × (戦法倍率% / 100) × 乱数(0.956〜1.044)

#### ステップ3: バフ/デバフ修正

同種バフ間、バフとデバフ間いずれも加算で合成。

#### ステップ4: 会心

会心発動時: × 1.5（デフォルト会心ボーナス）

### 計略ダメージ

#### ステップ1: 基礎値 (baseDmg)

baseDmg = max(最小保証, 最小保証 + hpDmg(兵力) + attrD(武勇差))

武勇差 : 攻撃側の知略 − 防御側の知略
attrD(差) : (0.0005 × 差² + 0.9 × 差 + 4.5) × 1.44  
hpDmg(兵力) : テーブル補間値（10,000兵力 → 387）
最小保証 : max(5, floor(兵力/1000) × 2)（10,000兵力 → 20）

#### ステップ2: 戦法ダメージ

戦法ダメ = baseDmg × (戦法倍率% / 100) × 乱数(0.956〜1.044)

#### ステップ3: バフ/デバフ修正

同種バフ間、バフとデバフ間いずれも加算で合成。

#### ステップ4: 奇策

奇策発動時: × 1.5（デフォルト会心ボーナス）

### 回復

#### ステップ1: 回復量の計算  

回復量 : (回復基礎値 + 依存ステータス) * (戦法倍率% / 100) * 乱数(0.956〜1.044)

回復基礎値 : 144.09In(兵士数) - 897.91

#### ステップ2: 負傷兵プールの制約

回復は負傷兵プール（injured）からしか回復できません。

実際の回復量 = min(計算回復量, injured の残量)

- ダメージを受けた時: injured += ダメージ × 90%、dead += ダメージ × 10%
- 死亡分（dead）は永続的に回復不可
- healBlock 状態（傷薬無効など）の場合: 回復量 0

#### ステップ3: 回復後HP

回復後HP = min(maxHp - dead, HP + 実際の回復量)

上限は maxHp - dead（死亡した兵士数分だけ最大HPが恒久的に減少）


### バフ・デバフ・発動確率
依存ステータスの記載があるバフ・デバフ・発動確率については、

反映値 = (依存ステータス-100)×0.25%×基礎値

という計算式が成り立つ

例1: 被ダメージ20%軽減、統率依存の場合
統率が200の場合は、被ダメージは25%軽減となる
統率が300の場合は、被ダメージは30%軽減となる

例2: 40%の確率で混乱を付与(知略依存)する場合
知略が200の場合は、50%の確率で混乱を付与する
知略が300の場合は、60%の確率で混乱を付与する
